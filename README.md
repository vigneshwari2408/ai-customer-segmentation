# AI Customer Segmentation System

## Objective
Build an AI-powered system to generate synthetic customer personas and segment them based on financial behavior.

## Approach
- Generated synthetic customer data using GPT-4
- Applied clustering techniques to identify user segments
- Interpreted clusters for business use cases

## Tools
Python, Azure ML, GPT-4, Clustering

## Output
Identified 3 key personas:
- Budget-conscious users
- High-value investors
- Credit-dependent customers

## Use Case
Helps financial institutions personalize products and improve targeting.

## Status
Hackathon project (BNP Paribas × CGI)
